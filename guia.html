se inicio con
npm init -y
despues instalar
npm install webpack webpack-cli --save-dev
"webpack": "^5.0.0-rc.3",
"webpack-cli": "^3.3.12",
"webpack-dev-server": "^3.11.0"
en ese orden despues configurar el archivo webpack.config.js
con el siguiten blok
const path = require('path');
module.exports = {
entry: './assets/javascript/entry.js',
output: {publicPath: '/',path: path.join(__dirname, '..'),filename: 'dist/javascript/bundle.js'},
plugins: [],
module: {}
};
despues insertar esto en package json en Script "start": "webpack-dev-server --open --config webpack.config.js"
esto creara un servidor local en el puerto 8080
para empezar con html instalamos npm install html-webpack-pluging revisar comentarios he inicio de configuraciones webpack para usar SAAS = una libreria para css
npm install style-loader css-loader sass-loader -- save dev luego instalar npm install node-sass-dev
luego se crea la carpeta css y el archivo css
